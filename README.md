# 🛍️ ShopEZ - Premium MERN E-Commerce Platform

ShopEZ is a full-stack, enterprise-grade e-commerce application built with the MERN stack (MongoDB, Express, React, Node.js). It features a modern, high-conversion UI inspired by top-tier platforms, complete with secure authentication, cart management, and a dedicated Dark Theme Admin Dashboard.

## ✨ Features
* **User Authentication**: Secure JWT-based login and registration.
* **Dynamic Product Catalog**: Real-time filtering by categories and search terms.
* **Smart Cart Engine**: Auto-calculates MRP, discounts, and total totals dynamically.
* **Order Tracking**: Users can view their past order history and current shipment status.
* **Executive Admin Console**: 
  * Add and manage inventory with live UI previews.
  * Update order fulfillment statuses (Pending, Shipped, Delivered).
  * Dashboard analytics for users, products, and orders.
* **UI/UX**: Responsive "Z-Pattern" layout, glassmorphism effects, and micro-interactions.

## 🛠️ Tech Stack
* **Frontend**: React (Vite), Context API, CSS Variables (Custom Design System)
* **Backend**: Node.js, Express.js
* **Database**: MongoDB (Mongoose ODM)
* **Security**: bcryptjs (Password Hashing), jsonwebtoken (JWT Sessions)
