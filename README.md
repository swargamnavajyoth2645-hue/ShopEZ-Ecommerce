# 🛍️ ShopEZ - Premium MERN E-Commerce Platform

ShopEZ is a full-stack e-commerce application built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). It provides a secure and responsive online shopping experience with user authentication, product browsing, shopping cart management, order processing, and a dedicated admin dashboard.

## ✨ Features
- **User Authentication** – Secure JWT-based login and registration.
- **Dynamic Product Catalog** – Browse products by category with search functionality.
- **Shopping Cart** – Add, update, and remove products with automatic price calculation.
- **Order Management** – Track orders and view purchase history.
- **Admin Dashboard**
  - Add and manage products.
  - Update product information.
  - Manage orders and inventory.
- **Responsive UI** – Modern interface optimized for desktop and mobile devices.

## 🛠️ Tech Stack
- **Frontend:** React (Vite), Context API, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas (Mongoose)
- **Security:** JWT Authentication, bcryptjs
